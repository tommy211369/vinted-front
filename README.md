## About the project
A fullstack Vinted web app clone : log in, sign up, filter and sort offers.  
Using my own API <a href="https://github.com/tommy211369/vinted-back" target="blank">(Vinted Back-End Repo)</a> , React Router, few Material UI elements (Range Slider, Spinner, Select).  

site : <a href="https://react-vinted-by-tommy.netlify.app/" target="blank">https://react-vinted-by-tommy.netlify.app/</a>
